# Pokemon Unbound PC Dex

An interactive Pokedex for Pokemon Unbound featuring 962+ Pokemon with complete evolution families.

## Live Demo

🌐 **[View Pokemon Unbound Dex](https://kukaratza.github.io/PokeUnboundDex/pokemon_unbound_dex.html)**

## Features

- **962+ Pokemon** - Complete collection with sprites
- **Evolution Families View** - Browse Pokemon organized by evolution chains with numbered PokeNumbers (1-534)
- **Grid View** - Classic card-based layout
- **Search & Filter** - Find Pokemon by name or Pokedex number
- **Regional Variants** - Filter by Alolan, Galarian, and Hisuian forms
- **Responsive Design** - Works on desktop and mobile devices

## Usage

Click on any Pokemon card to view detailed information on [UnboundWiki](https://unboundwiki.com).

## Evolution Families

The Evolution Families view organizes all Pokemon into 534 evolution groups, each marked with a unique PokeNumber badge. This includes:
- Complete evolution chains (e.g., Bulbasaur → Ivysaur → Venusaur)
- Branching evolutions (e.g., Eevee's multiple evolutions)
- Single-stage Pokemon (e.g., Legendaries)
